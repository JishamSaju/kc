Report Tracking

Award Report Tracking is accomplished in the KC system by using customizable Lookup screen functionality.

Access

Award Report Tracking is accessed from the Post-Award group on the Central Admin menu. 

Figure 745  Central Admin Menu > Post-Award Group > Award Report Tracking - Location

 

The lookup icon takes you to the Award Reporting Lookup screen.

 

Figure 746  Award Reporting Lookup Screen – Search Criteria Section Layout

 

It employs many tools that are standard to all KC Lookup screens.

General Lookup Reference:  For detailed information about the standard features and usage of Lookup screens in KC, see Lookup in Overview > User Interface Orientation > Selection, Entry & Command Tools.

The Award Reporting Lookup screen also utilizes features common to all document-specific Lookup screens.

Doc Search Reference:  For detailed information on basic, superuser, named & document-specific searches, see the Document-Specific Searches section of  Doc Search in Overview > User Interface Orientation.

Award Report Lookup allows for result narrowing via the use of wildcard criterium.

Wildcard Reference:  For more information about how to use wildcards to refine search records returned, see Searching With Wildcards in Overview.

 

Searching for an existing Award Report

If you are a Central Administrator with sufficient permission, you can search for existing Award Reports using the Award Report Tracking lookup in the Post-Award group on the Central Admin KC menu.

To track an Award Report:

1.

On the Central Admin menu, in the Post-Award group, click the Award Report Tracking lookup icon.

The Award Reporting Lookup screen appears.

2.

Optionally, refine your search as desired using the entry and selection tools in the search criteria section of the Award Reporting Lookup screen (for example, by selecting a Report Class).

3.

Click the search  button.

The results are displayed in a tabular format on the Award Reporting Lookup screen.

4.

Optionally, click the change view button to specify, customize and update the the view.  Options include PI, Award, or Sponsor optimized presets; as well as Group and Detail selections by field.

5.

Click the show  button to display all fields for a particular report result table row.

 End of activity.
