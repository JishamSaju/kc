Action Requested

Designates the type of action that is requested by an Action Request.  The action one needs to take on a document; also the type of action that is requested by an Action Request. Actions that may be requested of a user are:

Acknowledge - requests an acknowledgment of the viewing of a document

Approve - requests an approve or disapprove action

Complete - requests the user to enter additional information in a document so that the content of the document is complete

FYI - a notification to the user regarding the document, intended to simply makes a user aware of the document
