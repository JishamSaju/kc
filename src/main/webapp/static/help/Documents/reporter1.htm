Reporter

The Reporter section is a dynamic section of the e-doc based on the person who is accessing it and their role as it pertains to the disclosure.  For example, the initiator of the disclosure is considered to fill the role of Reporter.  This section has two subsections:  Contact Information and Unit Details that automatically retrieve and display information maintained elsewhere in the system according to the currently logged-in user who has acessed the e-doc.

Unit Details

The Unit Details are populated automatically based on the logged-in user, but may be modified.  So, in the case of multiple units, you have tools that give you the ability to search for, select and add additional unit line items, which are automatically numbered sequentially.  Additionally, for one line item only, you are able to specify which unit is considered the Lead unit for the project by checking a box.

Awards

The Award section of the Disclosure page displays Award and associated Financial Entities subsections for each Award document.  The Award Number in the subsection label, the Title and Award Date.  The Financial Entities subsection label shows the review status -for example, Financial Entities (01/ Reviews Complete), prefixed by the exclamation point icon ! which serves as an alert on the disclosure review progress.  The Financial Entities subsection displays the following four columns:

Proposals

The Proposals section of the Disclosure page is comprised of subsections for linked Proposals and Financial Entities.  The Proposals subsection label displays the Proposal Number, and the subsection displays fields of key identifying information about the proposal, including:

Proposal Name

Sponsor

Start Date

End Date

PI Name
