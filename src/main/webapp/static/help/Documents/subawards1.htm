Table 677  Maintenance Menu > Subawards Group – Link Descriptions

Link  (crossreferenced to subtopic)

Brief Description

Subaward Status

Allows for the definition of a list of states from which to select (for example, ‘Active’ or ‘Closed’).

Subaward Approval Type

Allows for the establishment and maintenance of various common types of approval required for subawards that can be selected from within the Subaward e-doc (for example, FAR 52.244-6).

Subaward Forms

Subcontract Forms may be uploaded as attachments and described for ease of lookup, reference, and download.

Closeout Type

Create and maintain a list of reasons for the closing out of subawards, which can be selected from within the Subaward e-doc (for example, ‘Audit Completed’).
