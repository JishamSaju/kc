Cross-Application Doc Search, Inquiry and Retrieval

  Effort Certification, Account, and Budget Adjustment documents that reside in the KFS system can be searched for and retrieved in KC search results using a Document Lookup screen.  Document ID links displayed  in result tables cause the document to be opened in the system in which it resides (KFS) in a new browser window, while your KC screen remains in the background in the original browser window.
