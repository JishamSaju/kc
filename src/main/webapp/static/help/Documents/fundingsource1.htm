Funding Source

The Funding Source page of the Subaward document is where you select the Award document that is to be associated with this Subaward document. 

When you select an Award Number, the remaining fields are automatically populated based on your selection, when the information is available in that record. 

The section employs standard line item functionality to add sequentially numbered rows to a table below the selection row in the case of multiple sources.

For more information about using the standard line item functionality in KC, see Common Line Item Operations in Overview > Common E-Doc Procedures.
