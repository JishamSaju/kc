Supplemental Online Resources

The following section offers a listing of links to official Kuali Coeus user documentation resources delivered with each major release of Kuali Coeus software, as written by the Kuali Coeus project staff and published by the Kuali Foundation, Inc., including the online help system that is a part of the "out-of-box" KC research administration software system.  Brief descriptions are provided for each resource. 

Help

Kuali Coeus Test Drive Online Help

https://testdrive.kc.kuali.org/kc-ptd/static/help/default.htm
KC knowledge base - searchable, indexed complete user doc set

Kuali Coeus Online Help Information Site

https://sites.google.com/a/kuali.org/kc-user-documentation/online-help-information

Help FAQs, how to customize KC OLH, Help project settings, KC help configuration for developers, inserting training videos into Kuali help, section 508 compliance in Kuali help, help style definitions

requires a Kuali Information System (KIS) account (anyone can create one at kuali.org/kis)

Wiki

Kuali Coeus Documentation and Implementation Wiki

https://wiki.kuali.org/display/KRADOC
KC public global wiki space for documentation and implementation
