Disclosure Actions

Data Validation is the only section of the Disclosure Actions page.  It is used to turn on or off the validation feature, which displays warning and error messages about the Disclosure e-doc.  When such messages exist, they are accompanied by fix buttons that allow you to go directly to the area of the Disclosure e-doc where the error or warning occurs, fix it, save it, then turn it on again to pass validation.
