Quicklinks Group

 

The Quicklinks group appears on the Researcher, Unit, and Central Admin menus.  The Researcher menu contains fewer links.  The purpose is to display convenient links for commonly-performed actions for each user type.  Primarily these include links to specialized Lookup screens that allow for efficient searching of existing e-doc records from the database tables.  They also include external system lookups (Grants.gov opportunities) and the ability to view documents that have been locked for editing.

Table 35 Quicklinks Group Link Descriptions

Link

Brief Description

Pessimistic Lock

Takes you to the Pessimistic Lock Lookup screen, which allows you to refine criteria and search for locks by document number; and take action on them (for example, delete) depending on the permissions associated with your role.  You are able to view the lock owner and date/time the lock was generated for a particular document.

Current & Pending Support

 

Grants.gov Opportunity Lookup

Takes you to the Opportunity Lookup screen, which allows you to search for Grants.gov opportunities by Opportunity ID or CFDA Number.

Non-Employee Lookup

Takes you to the Rolodex Lookup screen, which allows you to refine criteria and search for non-employee contact information.  Actions available from retrieved results include view, edit and copy depending on the permissions associated with your role.

Sponsor Lookup

Takes you to the Sponsor Lookup screen, which allows you to refine criteria and search for sponsor identifying and contact information.  Actions available from retrieved results include view, edit and copy depending on the permissions associated with your role.

Change Password

 

Keyword Lookup

Takes you to the Keyword Lookup screen, where you are able to search for keywords by code and/or description and view retrieved results.  Depending on the permissions associated with your role, additional available actions include create new, edit, and copy.
