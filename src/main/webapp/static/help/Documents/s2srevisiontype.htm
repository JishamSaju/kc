S2S Revision Type

 

Revision Type Document

Edit S2S Revision Type

The S2S Revision Type maintenance document is used to establish and maintain numeric codes and textual description values which are stored in database tables.  This data serves as reference information that is made available for display and selection in various areas of the system, including other e-docs.  An S2S Revision Type is a change in a sponsor's financial obligation or contingent liability of an existing award.  For example, the NIH uses the following revision types: (1) Increase award (2) Decrease award (3) Increase duration (4) Decrease duration and (5) Other.

Caution:  Revision types are pre-defined for S2S submission.  Do not change.
