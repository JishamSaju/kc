SubAward

The SubAward page of the Subaward document is the first one on the left and is by default the initial page displayed upon successful access.  It is used to record and display basic and key identifying information about the subaward record.  Several required fields must be populated with entries and/or selections in order to be able to navigate to subsequent pages and in order to save a draft of the document that can be completed at a later time.

It has the following two sections:

•      Document Overview

•      SubAward

Document Overview

The Document Overview section on the SubAward page of the KC SubAward e-doc is a section that appears on most e-docs.  It contains one field that is required – Description, which appears in doc search and action list table column cell display.  Since this is shared, it is documented in a single location.

 

For complete instructions about how to use this common e-doc section, see Document Overview in Overview > Common E-Doc Sections.
