Entity Name

Required.  Type into the field to enter a textual representation of this financial entity.  Consider the common reference to the entity to facilitate lookup ease.  The system will prevent creation of a duplicate active entity, so this must be unique.

  Sponsor Code

Enter the sponsor code associated with the financial entity. Upon successful validation of sponsor code by the system, the financial entity data fields will be automatically populated with data from the sponsor table.  The system returns the Sponsor ID number to the panel and displays the Sponsor name below the number. Type the suspected value in the box and click the direct inquiry icon or click the lookup icon to search for and return the value to populate the box automatically.

  Upon successful validation of the sponsor code entered by the user, the system automatically populates certain financial entity related fields with information from the sponsor table.  The following fields within the new financial entity will be overridden by corresponding data fields from the sponsor table, but conversely, may be subsequently edited:

-Entity name will be overridden with Sponsor name. 

-Entity type will be overridden by Sponsor type code. 

-Entity address will be overridden by Address Book Id that represents different address fields.

 

  Related To Organization Flag

Click within the checkbox to select the item.  A check mark appears within the checkbox to indicate the item is selected.  Click within the checkbox again to clear the mark and deselect the item.
