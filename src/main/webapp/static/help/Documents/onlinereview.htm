Online Review

 

The Online Review page of the Protocol document appears as an additional tabbed page to the right of the Protocol Actions page when the document has been submitted (Submitted to IRB status).  It allows you to specify a reviewer and a due date for a new online review of the protocol.  An explanation field gives you the ability to describe the purpose of the review.  After specifying this basic information and clicking the create button provided, the reviewer will receive the review request in their action list.

Table 253 Protocol Document, Online Review Page - Create New Online Review Section Field Descriptions

Field

Description

Reviewer

Required.  Select an individual person to be the recipient of the new online review action request.  Use the drop-down menu by clicking the down arrow to display the list, and then click on an item in the list to highlight and select it to populate the box with your selection.

Description

Required.  Enter identification text that serves as a brief summary of the purpose of the review.  Click within the text box (or press the tab  key from a previous field) to relocate the cursor to the field, and then type (or paste from virtual clipboard) to enter text in the box as necessary to provide the appropriate information.

Organization Document Number

Type the desired number for an associated document outside the KC system as necessary according to your institutional requirements.  Click within the text box (or press the tab  key from a previous field) to relocate the cursor to the field, and then type (or paste from virtual clipboard) to enter text in the box as necessary to provide the appropriate information.

Requested Date

Display-only.  The date the new online review request was created.

Due Date

Enter the date the new online review is due from the specified Reviewer.  Click the calendar icon to select, or type in mm/dd/yyyy format to populate the box with the desired date.

Explanation

Enter a textual explanation that expands on the Description in greater detail to make the review’s purpose clear.  Click within the text box (or press the tab  key from a previous field) to relocate the cursor to the field, and then type (or paste from virtual clipboard) to enter text in the box as necessary to provide the appropriate information.  Click the add note  icon to view/edit/paste text in a new browser window, then click the continue button to return to the text entry field in the document.  After saved, click the green arrow  symbol to view full text in a separate browser window.

 

Online Review Section

After filling out the required fields in the Create New Online Review section and clicking the create  button, a new tabbed section appears below it name Online Review:  First, Last (where First, Last displays the name of the Reviewer).

Figure 434 Protocol Document – Online Review Section Example

 

Table 254 Protocol Document, Online Review Section - Subsection Descriptions

Subsection

Description

Document Overview

Displays basic identifying information about the Protocol document.

Online Review

Allows you to specify a Due Date, online review Status, and Determination Recommendation, which can include one of the following options:  Substantive Revisions Required, Specific Minor Revisions, Approve, Expedited Approval, Grant Exemption, Expedited Approval, Close Administratively, Terminate, or Disapprove.

Review Comments

Select and add Standard Comment(s) or type to enter and add non-standard comment(s);  flag as Private or Final as desired.  Click the add button to add your selection/entry as a numbered line item.

Protocol Review Actions

Click the save , approve , disapprove  or reject  buttons as appropriate after making changes in this section.
