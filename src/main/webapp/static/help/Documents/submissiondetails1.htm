Submission Details

 

The Submission Details subsection of the Summary & History section includes five subsections:  Submission Details, Reviewers, Vote Summary, Check List Items, and Review Comments.  They allow you to view information pertaining to the review of the Protocol document.  The table below summarizes the purpose of each.

Figure 468 Protocol Document, Protocol Actions Page, Summary & History Section, Submission Details Subsection Layout

Submission Details

Includes the most general information about the Protocol that has been submitted to the committee.  It provides enough detail to allow for ease of identification of the submission.

Figure 469 Protocol Document, Protocol Actions Page, Summary & History Section, Submission Details Subsection – Submission Details Example

Reviewers

The PI is not able to view the Reviewer names and types, however, IRB Administrators are able to view them.

Figure 470 Protocol Document, Protocol Actions Page, Summary & History Section, Submission Details Subsection – Reviewers Example

Vote Summary

Allows you to view the number of votes by type.  Click the green arrow icon to view comments.

Figure 471 Protocol Document, Protocol Actions Page, Summary & History Section, Submission Details Subsection – Vote Summary Example

Check List Items

When pending review work to be accomplished by the committee is identified and entered, it appears here as numbered line items.

Figure 472 Protocol Document, Protocol Actions Page, Summary & History Section, Submission Details Subsection – Check List Items Example

Review Comments

Predefined comments selected by the committee members are displayed.  Private indicates the view is restricted to members only.  Final indicates if appropriate action is taken by the PI as a result of reading the comment, no further comment or revision of the comment will occur.

Figure 473 Protocol Document, Protocol Actions Page, Summary & History Section, Submission Details Subsection – Reviewer Comments Example

Review Attachments

After a file has been attached in the Manage Review Attachments subsection of the Request an Action section, prior to submission, the ‘Protocol personnel can view’ option can be checked.  After submission, a system message appears in the notification area as follows:  “Protocol action Manage Review Attachments successfully completed.”  When the status of the Protocol document is submitted to workflow (and its status is Submitted to IRB), the Review Attachments subsection appears in the Summary & History section under Submission Details.

 

•      In the Submission Navigation area, click the show button in the Review Attachments subpanel.

The contents are shown in a table format.  This includes a numbered line item for each attached file, and displays the filename, time/date/user for last updated and created, as well as an Action column.

 

•      Click the view  button in the Action column to Open and/or Save the attached file to be reviewed.

Figure 474  Protocol Document > Protocol Actions Page > Summary & History Section > Submission Details Subsection > Review Attachments subpanel – View Action Button Location

Your browser will display a dialog box (popup window) which prompts you to either ‘Open with’ and select an appropriate application to open the file with (for example, Notepad or Excel) or ‘Save’ to save it to your local hard drive.
