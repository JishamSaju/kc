Modular Budget Overview

The Modular Budget Overview section of the Modular Budget page displays key dates and information about the requested cost based on your selection of a particular period.

Figure 723  Award Budget Document > Modular Budget Page > Modular Budget Overview (Period #) Example

 

Table 410 Award Budget Document > Modular Budget Page > Modular Budget Overview (Period #) – Field Descriptions

Field

Description

Project Start Date

The start date of the specific budget period being viewed.

Project End Date

The end date for the displayed period.

Project Total Requested Cost

Sum of all period total requested cost amounts from each budget period, including direct costs and F&A costs.
